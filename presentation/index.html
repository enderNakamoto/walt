Origin Story

The Problem We Hit

We were building a niche prediction market on Soroban. When it came time to test the frontend, we hit a wall — there was no way to programmatically mock Stellar wallets in Playwright tests.

So We Built This

stellar-wallet-mock — an open-source library that mocks the Freighter wallet in Playwright tests. Install it, pass a secret key, and your tests can sign Soroban transactions.

☆ github.com/SentinelFi/stellar_wallet_mock

Then We Thought Bigger

What if we didn't just mock the wallet — what if an AI agent could explore any Stellar dApp, generate the tests, and self-heal when things break?

That's WALT. It uses stellar-wallet-mock under the hood to launch autonomous front-end agents that find bugs before your users do.

1. Explore

Agent crawls the dApp with Playwright + wallet mock. Screenshots every page, extracts DOM, builds accessibility tree.

2. Inspect & Chat

Describe your test in plain English. Agent inspects live pages, asks about success criteria, verifies selectors before writing code.

3. Generate

Produces Playwright tests with smart waits, wallet mock, and verified selectors. Each step is a separate test block.

4. Run & Self-Heal

Executes tests, retries transient failures, and uses Claude vision to diagnose and fix structural failures automatically.

Self-Healing Tests

Your frontend changes. Your tests don't break.

With WALT: The agent sees the screenshot, notices TVL moved, updates the selector automatically, test passes. Zero human effort.
